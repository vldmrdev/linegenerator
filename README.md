### LLGEN is the Log Line Generator

This is a simple line-by-line logs generator for testing your scripts and algorithms or other stuff if you need.

#### Status: in development

### Install

```pip install llgen```


### Requirements 
- Python >= 3.9
- click >= 8.0.0
